## Architecture
```
n8n workflow
 └─ HTTP Request POST /run ─────────────────────────► initiator Lambda (30s)
 │ │ creates DynamoDB job
 │ ◄── 202 { jobId, statusUrl } ───────────────────── │ invokes worker async
 │ │
 │ n8n Wait node (waits up to 20 min for callback) ▼
 │ worker Lambda (900s)
 │ │ clones GitHub repo
 │ │ runs agent (cn or claude)
 │ │ git add / commit / push
 │ │ uploads files to S3
 │ │ marks job "completed"
 └─ ◄── POST callback_url { response, ... } ── │
 ▼
 DynamoDB ◄─── GET /status/{jobId}
```
